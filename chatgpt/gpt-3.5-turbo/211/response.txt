<<< Before refactoring:
```
Signature signature = Signature.getInstance("SHA256withRSA");
signature.setParameter("key", privateKey);
signature.setParameter("format", "DER");
```

>>> After refactoring:
```
Signature signature = Signature.getInstance("SHA256withRSA");
signature.setParameter(Signature.Parameter.KEY, privateKey);
signature.setParameter(Signature.Parameter.FORMAT, "DER");
```

In the refactored code, we're using the `setParameter` method with the `Signature.Parameter` enum instead of passing in a string for the parameter name. This is the recommended way to use the method now that the old way is deprecated.
