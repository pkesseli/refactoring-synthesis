Sure, here's an example refactoring:

<<< Before refactoring:
```
import java.awt.Component;
import java.awt.Dimension;

public class MyComponent extends Component {
    public void resize(Dimension d) {
        setSize(d);
    }
}
```

>>> After refactoring:
```
import java.awt.Component;
import java.awt.Dimension;

public class MyComponent extends Component {
    @Override
    public void setSize(Dimension d) {
        super.setSize(d);
    }
}
```

In the refactored code, we've replaced the deprecated `resize` method with an overridden `setSize` method. This is the recommended replacement according to the deprecation comment in the Javadoc.
